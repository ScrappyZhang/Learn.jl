Welcome to Learn's documentation!
=================================

*Learn.jl* is a Julia package for machine learning. It provides a common API for various types of machine learning algorithms. *Learn.jl* is inspired by sklearn_ and emulates its approach to unify various machine learning algorithms. The overarching goal of *Learn.jl* is to be the best API for machine learning in Julia by integrating Julia's machine learning packages in a common framework.

    .. _sklearn: http://scikit-learn.org
